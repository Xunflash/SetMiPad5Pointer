# SetMiPad5Pointer

> 替换小米平板圆点鼠标指针

#### 1.安装

安装release页面的apk，并在lsposed管理器中勾选系统框架即可使用

#### 2.介绍

通过hook系统框架中的MiuiMagicPointerService，修改被MIUI团队魔改过的圆形鼠标指针，提升鼠标操控时的定位精度
